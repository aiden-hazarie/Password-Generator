## Password Generator
Description: A program that creates strong, random passwords to keep online accounts safe. Instead of using easy-to-guess words (like "password123"), it mixes letters, numbers, and symbols to generate a secure password that hackers can’t easily crack.

## Key Features
- Customizable length and complexity (e.g., 12+ characters, mixed symbols).
- Avoids weak or repetitive patterns.
- Helps protect against cyber attacks like brute-force hacking.
